Para comenzar a programar lo primero que debemos hacer es descargar nuestro editor de código, yo recomiendo:

Visual Studio Code

Un editor de código potente y fácil de usar.

¿Cómo lo obtengo?

  1. Entrar a la página oficial y descargarlo para tu sistema operativo
  2. Instalarlo en tu computadora
  3. ¡Listo, ya podés empezar a codear!

Una vez instalado podes agregarle las siguientes extensiones.

  • Live Server: Sirve para previsualizar tu página en tiempo real en la web
  • Prettier: Formatea tu código para que siempre este ordenado.
